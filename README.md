# 🌟 Resume.io — Personal Resume Website

一個現代、簡潔、可立即部署的 **個人履歷網站**。  
使用 **HTML + CSS + JavaScript** 打造，適合求職、展示作品、建立個人品牌。

---

## 🔗 Demo  

👉 https://wayne0917.github.io/Resume.io/

---

## ✨ Features — 功能特色

- 🎨 乾淨俐落的版面設計  
- 📱 支援 RWD（手機/平板完美呈現）  
- ⚡ 純前端、免後端即可部署  
- 🧩 程式碼結構簡單易維護  
- 🎯 可自由擴充功能（Dark Mode、動畫、SPA 等）

## 🛠 Tech Stack — 使用技術

| 技術 | 說明 |
|------|------|
| ⚡ **HTML5** | 網站結構、語意化標籤 |
| 🎨 **CSS3** | 排版、配色、響應式設計 |
| 🔧 **JavaScript** | DOM 操作與互動效果 |
| 🐙 **Git / GitHub** | 版控 + Pages 部署 |
